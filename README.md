# Keithley-2001-oled

This is my attempt to put a OLED display in a Keithley 2001 - 2400 - 7000
This is for the almost unobtainable dot matrix VFD display.

At this moment only tested on a raspberry PI and raspberry PI pico
with a Keithley 2001 meter.

For the font I have added all the display test characters and also the bargrap characters.
